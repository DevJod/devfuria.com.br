## Instalação - php

Você precisará ter insalado:

+ Apache
+ PHP
+ Composer

No terminal, execute...

    git clone https://github.com/flaviomicheletti/devfuria.com.br
    cd devfuria.com.br
    composer install
    // dependendo da sua instalação, pode ser...
    ~/composer.phar install

Acesse `http://localhost/devfuria.com.br`.
