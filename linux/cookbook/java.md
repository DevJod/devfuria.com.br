### Debian-wheezy

	apt-get install openjdk-7-jre


### Centos 6.5+

    su -c "yum install java-1.7.0-openjdk"


### FEDORA 14+

	
	# su -c "yum install java-1.7.0-openjdk"


### UBUNTU 12+

Via software-center é mais fácil, procure por

	"OpenJDK" ou
	"openJavaX" (onde x é a versão mais recente).
