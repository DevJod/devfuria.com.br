No capítulo anterior nos conhecemos o HTML, neste iremos introduzir o CSS.

Enquanto o HTML se encarrega de estruturar o documento,
o CSS fica encubido de aplicar estilização ao documento. 

HTML -> estrutura
CSS  -> estilização

O que significa CSS ?

Cascading Style Sheet = Folha de Estilo em Cascata

É o código responsável pela estilização do documento.

O CSS pode se juntar ao HTML de 3 formas diferentes, uma delas é na forma de um arquivo.

O arquivo CSS nada mais é que um arquvo de texto plano nomeado com a extensão `.css`, por exemplo:

exemplo.css

Mas e esse tal de "cascata" ?
