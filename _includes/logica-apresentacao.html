Rápida apresentação

Este artigo compõem a série de exercícios da seção lógica de programação do site devfuria.

Ele é diferente de tudo o que você já viu ou ouviu falar sobre curso de lógica da programação.

Nós unimos o poder dos testes unitários aos exercícios de lógica,

Além disso, utilizamos 3 linguagens de programação: C, Python e JavaScript (Node.js).

Doravante, vamos direto para a prática e não perdemos tempo com teorias sobre lógica pois você as encontrará na web com facilidade e abundância.

Talvez você precise ler os artigos introdutórios listados na index Lógica de Programação, mas caso precise de ajuda, utilize os comentários no final do artigo.
