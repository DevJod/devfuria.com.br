Endentação
---

Em Python devemos endentar corretamente o código fonte. O interpretador depende da endentação.

Abaixo vemos um código endentado de forma errada:

    def spam():
    eggs = 12
    return eggs

    print spam()

Abaixo, vemos o mesmo código corrigido:

    def spam():
        eggs = 12
        return eggs

    print spam()
